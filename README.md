**Course:** Informatics for Telecommunications

**Team:** NetCore Team

## Project Description
Software-based centralized routing system composed of a controller application (Control Plane) and multiple router applications (Data Plane). 
The controller computes optimal routes using Dijkstra's algorithm and distributes forwarding tables to each router via TCP with TLS encryption.
